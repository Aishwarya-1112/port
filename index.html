About

I’m a frontend developer passionate about creating visually appealing and interactive web experiences. I specialize in using HTML to structure content, CSS to style it beautifully, and JavaScript to bring websites to life with dynamic functionality. I have experience with React, which allows me to build efficient and scalable user interfaces using reusable components. Ensuring that websites are responsive and user-friendly on all devices is a key part of my work. I’m proficient in integrating APIs to fetch data seamlessly and optimizing website performance for fast load times. I’m also skilled in using Git for version control and collaborating with teams to bring ideas to life. I stay up to date with the latest technologies and trends to continually improve my craft. Working closely with designers, I transform concepts into fully functional, high-quality websites. I strive to make sure every project I work on delivers an exceptional user experience across all browsers and devices.
